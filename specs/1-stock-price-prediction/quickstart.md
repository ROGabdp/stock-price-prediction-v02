# 快速入門：股價漲跌機率預測系統

**功能分支**: `1-stock-price-prediction`  
**日期**: 2025-11-12
**規格書**: [./spec.md](./spec.md)
**計畫**: [./plan.md](./plan.md)

本文件提供如何快速啟動並使用股價漲跌機率預測系統的指南。

## 1. 環境設定

確保您的系統已安裝 Python 3.12。建議使用虛擬環境來管理專案依賴。

```bash
# 建立虛擬環境
python -m venv venv

# 啟用虛擬環境 (Windows)
.\venv\Scripts\activate

# 啟用虛擬環境 (macOS/Linux)
source venv/bin/activate

# 安裝專案依賴 (待專案結構確定後，將提供 requirements.txt)
# pip install -r requirements.txt
```

**注意**: 具體的依賴套件將在專案結構確定後提供。目前已知的主要依賴包括：
- 機器學習: TensorFlow/PyTorch
- Web 框架: Flask
- UI 框架: Dash
- 繪圖: Plotly
- 資料處理: Pandas

## 2. 啟動後端服務 (Flask)

後端服務負責處理資料上傳、模型訓練、模型管理和預測邏輯。

```bash
# 進入專案根目錄
cd your-project-root

# 啟動 Flask 應用 (假設應用入口為 app.py)
# flask run
```

**注意**: 具體的啟動命令將根據實際的 Flask 應用結構而定。

## 3. 啟動前端介面 (Dash)

前端介面負責提供使用者互動、資料視覺化和結果展示。

```bash
# 進入專案根目錄
cd your-project-root

# 啟動 Dash 應用 (假設應用入口為 app.py)
# python app.py
```

**注意**: 具體的啟動命令將根據實際的 Dash 應用結構而定。

## 4. 使用系統

一旦後端和前端服務都已啟動，您可以在瀏覽器中訪問前端介面 (通常是 `http://127.0.0.1:8050/` 或類似位址)。

### 步驟:

1.  **上傳或選擇歷史資料**:
    *   在介面上找到資料選擇區域，上傳您的 CSV 格式歷史股價資料，或從現有列表中選擇。
2.  **訓練模型**:
    *   選擇資料集後，設定預測天數 N (例如 5 天)。
    *   點擊「開始訓練」按鈕。系統將自動進行參數調整並訓練模型。
3.  **查看預測結果**:
    *   訓練完成後，介面將顯示預測結果，並在圖表上展示歷史數據和未來 N 天的預測趨勢。
4.  **選擇已訓練模型**:
    *   您可以從模型列表中選擇不同的已訓練模型，以查看其預測結果。

## 5. 持續更新資料

系統將提供介面讓您輕鬆上傳新的歷史資料，以供模型持續學習和更新。
